# Prompt Library

The persistent prompt files live in:
`<SAFE_ROOT>\prompts`

## Files
- `controller-change-superprompt.md`: main prompt template for controller code changes with autonomous release follow-through and configuration awareness.
- `autopilot-hardening.prompt.md`: improve resilience and remove operational gaps.
- `autopilot-recovery.prompt.md`: recover a broken release cycle safely.
- `autopilot-self-check.prompt.md`: validate the persistent installation end to end.

Use the prompt library when the user wants ready-made wording or when you need to bootstrap a new chat with the established flow quickly.
