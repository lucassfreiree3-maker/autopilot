# BBDevOpsAutopilot Portable Kit

This directory contains a portable copy of the operational documentation, prompt library, and Codex skill references needed to explain and reinstall the controller release flow on another machine or in another AI context.

## Not included
- Encrypted GitHub token
- Canonical controller clone contents
- Runtime state and reports

## Included
- Manifest copy
- Docs
- Prompts
- Skill copy
- Bootstrap installer script
